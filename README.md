# Demo_pybind11

## 环境说明
- Visual Studio 2022
- pybind11 v2.13.5
- python3.8.10, 只安装必要的内容, 然后拷贝到此目录, 需要重新装下[pip](https://www.cnblogs.com/yiwenrong/p/12659336.html), 不然无法直接使用pip, 不重装只能`python -m pip`

## 注意

`set PYTHONHOME=D:\Work\DemoPybind11\3rd\python38`
